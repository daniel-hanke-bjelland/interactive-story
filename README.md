# interactive-story

Du kan navigere mellom scenene med forksjellige tastaturtaseter. noen tar deg tilbake, andre tar deg frem. Det er en timeout, sånn at du må vente så og så lenge før du kan gå videre, og du har en maxtid på scenen, før du blir automatisk sendt videre. Pluss mye mer, men dette var bare litt kort info. 

Kos deg med den iteraktive historien!!
